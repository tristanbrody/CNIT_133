For an example of an API, I chose one that is free, fun and easy to use: Deck of Cards API

    Fun facts about the API:

  • Main URL above has the docs
  • It does not require any developer key to acces
  • Responses are provided in JSON
  • All interactions with the API can be made via get requests, with your parameters included as query strings added to the endpoint
  • To get started, send a get request to https://deckofcardsapi.com/api/deck/new/shuffle/?deck_count=1 to get a new deck ID
  • Once you have a deck ID, you can pass that deck ID into endpoint https://deckofcardsapi.com/api/deck/deck_id>/draw/?count=1 to draw one or more cards
